Usability Study: Findings

1

Menu icon

Users got confused that the menu icon is a book, so added a letter under the icon.

2

Letter balance

The user got confused about which information is most important since I designed the same letter size for a description of the item.

3

Confirm

User couldn't make sure the address and billing address after the confirm page

Accessibility considerations

1

I used headings with different sized text for clear visual hierarchy.

2

On the button, use shadow and different colors with the cancel button and the next, confirm button. Therefore, users are distinguished and avoid mistaken selection.

3

Used hyperlinks on the payment page, so it is easier to read although there is surrounded by no rectangle.

Takeaways

Impact:

Our target users shared that the design was friendly and simple, yet some of my design was not clear or make confusion due to a lack of information.

What I learned:

What I learned through the project is, that even small thing such as the location where I design, makes user confused and can't reach the next page. So I need to remember that the users are first.
